# Design Questions

This file collects decisions that need to be made before moving into a more
detailed systems design.

## Specializations

- How strictly should specializations be limited so trade is necessary but the game is not frustrating?
- Can specializations be reset?
- If resets are possible, what should they cost?
- When should the player make their first hard specialization decision?

## Economy

- Should each server have a local economy?
- Should the market be global across servers?
- How can monopolies be prevented?
- How can price manipulation be prevented?
- Should prices be fully controlled by players?

## First Minutes Of The Game

- What do the first 5 minutes of gameplay look like?
- When does the player first encounter trade?
- When does the player understand that they should not do everything alone?
- What is the first product the player can sell to another player?

## Products

- Can products have quality?
- Can products carry a producer brand?
- Should a product store information about its creator?
- Can different producers create the same item at different quality levels?

## NPC Workers

- How do NPC workers work?
- Can workers produce while the player is offline?
- Do workers have their own skills?
- Do workers consume food, tools, or wages?
