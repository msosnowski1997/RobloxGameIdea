# Roblox Game Idea

Dokumentacja koncepcji gry ekonomiczno-rzemieślniczej w Roblox.

## Koncepcja W Jednym Zdaniu

Gracz buduje wyspecjalizowane miasto-rzemiosło, produkuje dobra, handluje z innymi
graczami i zdobywa reputację jako konkretny producent w gospodarce tworzonej przez
społeczność.

## Najważniejsza Teza

To nie jest klasyczny tycoon o powiększaniu bazy.

Miasto jest narzędziem produkcji. Prawdziwym celem jest zbudowanie pozycji w
gospodarce serwera: bycie znanym dostawcą mebli, stali, jedzenia, narzędzi,
luksusowych przedmiotów albo innych dóbr.
